Jan is an open-source desktop application that serves as a privacy-focused alternative to ChatGPT, allowing users to download and run large language models (like Llama, Gemma, and Qwen) entirely offline on their local machines while also supporting cloud integrations with providers like OpenAI and Anthropic. The key feature for my project is that Jan exposes an OpenAI-compatible API server at localhost:1337, which means other applications can connect to it programmatically.
